# AICopy 视频接口文档

Base URL:

```text
https://api.aicopy.top
```

认证:

```http
Authorization: Bearer <API_KEY>
Content-Type: application/json
```

Curl 示例统一写法:

```bash
export AICOPY_API_KEY="你的 API Key"
```

## 1. 查询模型

```http
GET /v1/models
```

请求示例:

```bash
curl https://api.aicopy.top/v1/models \
  -H "Authorization: Bearer $AICOPY_API_KEY"
```

视频相关模型分组:

```text
firefly-runway45
firefly-sora2
firefly-veo31
sd2
```

即梦 / SD2 当前可用模型:

```text
sd2-720p-fast
sd2-720p
sd2-1080p-fast
sd2-1080p
sd2-720p-min-fast
sd2-720p-min
```

## 2. 通用视频生成：Chat Completions

适用于 Firefly / Sora / VEO / Runway 类视频模型。

```http
POST /v1/chat/completions
```

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| model | string | 是 | 视频模型 ID |
| messages | array | 是 | OpenAI chat 格式内容 |
| stream | boolean | 建议 | 建议传 true，流式返回视频地址 |
| aspect_ratio | string | 否 | 宽高比，如 16:9、9:16、1:1、21:9 |
| video_length | number | 否 | 视频秒数，如 4、5、6、8、10、12 |
| resolution | string | 否 | 清晰度，如 720p、1080p，部分 Sora 模型可传 "-" |

### messages.content 参数

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| type | string | text 或 image_url |
| text | string | 提示词，type=text 时使用 |
| image_url.url | string | 图片 URL 或 data:image/...;base64,... |

### 文生视频示例

```json
{
  "model": "firefly-veo31-6s-16x9-720p",
  "messages": [
    {
      "role": "user",
      "content": [
        {
          "type": "text",
          "text": "未来城市上空，一艘银色飞船穿过云层，电影级光影"
        }
      ]
    }
  ],
  "stream": true,
  "aspect_ratio": "16:9",
  "video_length": 6,
  "resolution": "720p"
}
```

```bash
curl https://api.aicopy.top/v1/chat/completions \
  -H "Authorization: Bearer $AICOPY_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "firefly-veo31-6s-16x9-720p",
    "messages": [
      {
        "role": "user",
        "content": [
          {
            "type": "text",
            "text": "未来城市上空，一艘银色飞船穿过云层，电影级光影"
          }
        ]
      }
    ],
    "stream": true,
    "aspect_ratio": "16:9",
    "video_length": 6,
    "resolution": "720p"
  }'
```

### 图生视频 / 首帧生视频

```json
{
  "model": "firefly-veo31-6s-16x9-720p",
  "messages": [
    {
      "role": "user",
      "content": [
        {
          "type": "image_url",
          "image_url": {
            "url": "data:image/png;base64,xxxx"
          }
        },
        {
          "type": "text",
          "text": "让画面中的人物缓慢转身，背景灯光缓慢闪烁"
        }
      ]
    }
  ],
  "stream": true,
  "aspect_ratio": "16:9",
  "video_length": 6,
  "resolution": "720p"
}
```

### 首尾帧 / 多参考图

把多张图片依次放进 content，再追加 text:

```json
{
  "type": "image_url",
  "image_url": {
    "url": "data:image/png;base64,首帧"
  }
},
{
  "type": "image_url",
  "image_url": {
    "url": "data:image/png;base64,尾帧"
  }
},
{
  "type": "text",
  "text": "从第一张自然过渡到第二张，保持人物一致"
}
```

### 流式返回解析

返回为 SSE:

```text
data: {"choices":[{"delta":{"content":"生成中..."}}]}
data: {"choices":[{"delta":{"content":"https://example.com/video.mp4"}}]}
data: [DONE]
```

客户端从以下位置提取视频地址:

```text
choices[0].delta.content 中的 http/https URL
<video src="...">
<source src="...">
普通文本里的 https://xxx.mp4
```

## 3. 即梦 / SD2 视频生成：异步任务

适用于模型:

```text
sd2-720p-fast
sd2-720p
sd2-1080p-fast
sd2-1080p
sd2-720p-min-fast
sd2-720p-min
```

提交任务:

```http
POST /v1/videos
```

查询任务:

```http
GET /v1/videos/{task_id}
```

下载结果:

```http
GET /v1/videos/{task_id}/content
```

### SD2 JSON 请求参数

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| model | string | 是 | 即梦 / SD2 模型 |
| prompt | string | 是 | 视频提示词 |
| seconds | string | 建议 | 秒数字符串，如 "5" |
| duration | number | 建议 | 秒数，如 5 |
| video_length | number | 建议 | 秒数，如 5 |
| size | string | 是 | 输出尺寸，如 1280x720 |

同一个秒数建议同时传 `seconds`、`duration`、`video_length`，兼容性最好。

### SD2 文生视频示例

```json
{
  "model": "sd2-720p-fast",
  "prompt": "一个女孩站在海边，夕阳逆光，头发被风吹动，电影感镜头",
  "seconds": "5",
  "duration": 5,
  "video_length": 5,
  "size": "1280x720"
}
```

```bash
curl https://api.aicopy.top/v1/videos \
  -H "Authorization: Bearer $AICOPY_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "sd2-720p-fast",
    "prompt": "一个女孩站在海边，夕阳逆光，头发被风吹动，电影感镜头",
    "seconds": "5",
    "duration": 5,
    "video_length": 5,
    "size": "1280x720"
  }'
```

### SD2 图生视频：multipart/form-data

有参考图时，用 multipart 上传文件。

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| model | form field | 是 | 模型 |
| prompt | form field | 是 | 提示词 |
| seconds | form field | 建议 | 秒数字符串 |
| duration | form field | 建议 | 秒数 |
| video_length | form field | 建议 | 秒数 |
| size | form field | 是 | 输出尺寸 |
| input_reference | file | 否 | 第 1 张参考图 |
| input_reference_2 | file | 否 | 第 2 张参考图 |
| input_reference_3 | file | 否 | 第 3 张参考图 |
| input_reference_4 | file | 否 | 第 4 张参考图 |

首帧示例:

```bash
curl https://api.aicopy.top/v1/videos \
  -H "Authorization: Bearer $AICOPY_API_KEY" \
  -F "model=sd2-720p-fast" \
  -F "prompt=让图片中的人物缓慢转身，镜头轻微推进" \
  -F "seconds=5" \
  -F "duration=5" \
  -F "video_length=5" \
  -F "size=1280x720" \
  -F "input_reference=@ref1.png"
```

首尾帧示例:

```bash
curl https://api.aicopy.top/v1/videos \
  -H "Authorization: Bearer $AICOPY_API_KEY" \
  -F "model=sd2-720p" \
  -F "prompt=从第一张自然过渡到第二张，保持人物一致" \
  -F "seconds=5" \
  -F "duration=5" \
  -F "video_length=5" \
  -F "size=1280x720" \
  -F "input_reference=@first.png" \
  -F "input_reference_2=@last.png"
```

### SD2 size 参数

| 宽高比 | 720p 模型 | 1080p 模型 |
| --- | --- | --- |
| 16:9 | 1280x720 | 1920x1080 |
| 9:16 | 720x1280 | 1080x1920 |
| 1:1 | 720x720 | 1080x1080 |
| 4:3 | 720x720 | 1080x1080 |
| 3:4 | 720x720 | 1080x1080 |

插件里的尺寸判断逻辑是：模型名包含 `1080p` 就用 1080，否则用 720。

### SD2 任务提交返回

提交后从这些字段中取任务 ID:

```text
id
task_id
video_id
data.id
data.task_id
```

如果提交时直接返回视频 URL，则从这些字段取:

```text
output.url
output.video_url
output.download_url
result_url
video_url
url
download_url
file_url
```

### SD2 轮询状态

```bash
curl https://api.aicopy.top/v1/videos/TASK_ID \
  -H "Authorization: Bearer $AICOPY_API_KEY"
```

成功状态:

```text
completed
succeeded
success
```

失败状态:

```text
failed
cancelled
canceled
error
```

处理中状态通常继续轮询:

```text
queued
submitted
processing
running
in_progress
```

建议轮询间隔:

```text
5 秒
```

建议总超时:

```text
1200 到 1800 秒
```

### SD2 下载

如果轮询返回了视频 URL，直接 GET 下载。

如果任务成功但没有 URL，下载:

```http
GET /v1/videos/{task_id}/content
```

示例:

```bash
curl https://api.aicopy.top/v1/videos/TASK_ID/content \
  -H "Authorization: Bearer $AICOPY_API_KEY" \
  -o output.mp4
```

### SD2 min 模型限制

适用于:

```text
sd2-720p-min-fast
sd2-720p-min
```

限制:

```text
时长不能低于 5 秒
参考图最多 4 张
首尾帧模式必须正好 2 张图
```

## 4. 旧版星链云兼容接口

如果 Base URL 不是 `api.aicopy.top` 或 `vjimeng.vip`，插件会走旧接口。

提交:

```http
POST /v1/video/submit/generate
```

查询:

```http
GET /v1/video/fetch/{task_id}
```

### 旧版提交参数

```json
{
  "model": "sd2-720p-fast",
  "prompt": "提示词",
  "duration": 5,
  "metadata": {
    "ratio": "16:9",
    "enableSound": "off",
    "modeType": "mixed2video"
  },
  "images": [
    "data:image/jpeg;base64,xxxx"
  ]
}
```

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| model | string | 是 | 模型 |
| prompt | string | 是 | 提示词 |
| duration | number | 是 | 秒数 |
| metadata.ratio | string | 是 | 宽高比 |
| metadata.enableSound | string | 否 | off 或 on |
| metadata.modeType | string | 图生时建议 | mixed2video 或 frames2video |
| images | array | 否 | data URL 图片数组 |

`modeType`:

```text
mixed2video   全能参考 / 多图参考
frames2video  首尾帧
```

## 5. 模型参数速查

### Firefly Sora2

模型格式:

```text
firefly-sora2-{秒数}s-{比例}
firefly-sora2-pro-{秒数}s-{比例}
```

可用秒数:

```text
4, 8, 12
```

可用比例:

```text
16x9, 9x16
```

示例:

```text
firefly-sora2-4s-16x9
firefly-sora2-8s-9x16
firefly-sora2-pro-12s-16x9
```

### Firefly VEO3.1

模型格式:

```text
firefly-veo31-{秒数}s-{比例}-{清晰度}
firefly-veo31-fast-{秒数}s-{比例}-{清晰度}
firefly-veo31-ref-{秒数}s-{比例}-{清晰度}
```

可用秒数:

```text
4, 6, 8
```

可用比例:

```text
16x9, 9x16
```

可用清晰度:

```text
720p, 1080p
```

示例:

```text
firefly-veo31-6s-16x9-720p
firefly-veo31-fast-8s-9x16-1080p
firefly-veo31-ref-4s-16x9-720p
```

### Firefly Runway 4.5

模型格式:

```text
firefly-runway45-{秒数}s-{比例}-720p
```

可用秒数:

```text
5, 10
```

可用比例:

```text
21x9, 16x9, 4x3, 1x1, 3x4, 9x16, 9x21
```

示例:

```text
firefly-runway45-5s-16x9-720p
firefly-runway45-10s-1x1-720p
firefly-runway45-10s-21x9-720p
```

### 即梦 / SD2

模型:

```text
sd2-720p-fast
sd2-720p
sd2-1080p-fast
sd2-1080p
sd2-720p-min-fast
sd2-720p-min
```

常用参数:

```text
duration: 4 到 15 秒，min 模型最低 5 秒
aspect_ratio: 16:9, 9:16, 4:3, 3:4, 1:1
enable_sound: off, on
generation_mode: 文生视频, 全能参考, 首尾帧
modeType: mixed2video, frames2video
```

## 6. Python 参考封装

```python
import time
import requests

BASE_URL = "https://api.aicopy.top"
API_KEY = "你的 API Key"

headers = {"Authorization": f"Bearer {API_KEY}"}

def submit_sd2(prompt, model="sd2-720p-fast", duration=5, size="1280x720"):
    payload = {
        "model": model,
        "prompt": prompt,
        "seconds": str(duration),
        "duration": duration,
        "video_length": duration,
        "size": size,
    }
    r = requests.post(
        f"{BASE_URL}/v1/videos",
        headers={**headers, "Content-Type": "application/json"},
        json=payload,
        timeout=120,
    )
    r.raise_for_status()
    data = r.json()
    return data.get("id") or data.get("task_id") or data.get("video_id") or data.get("data", {}).get("task_id")

def poll_video(task_id, timeout=1800, interval=5):
    start = time.time()
    while time.time() - start < timeout:
        r = requests.get(f"{BASE_URL}/v1/videos/{task_id}", headers=headers, timeout=30)
        if r.status_code != 200:
            time.sleep(interval)
            continue
        data = r.json()
        status = str(data.get("status", "")).lower()
        if status in {"completed", "succeeded", "success"}:
            for key in ("result_url", "video_url", "url", "download_url", "file_url"):
                if data.get(key):
                    return data[key]
            output = data.get("output") or {}
            for key in ("url", "video_url", "download_url"):
                if output.get(key):
                    return output[key]
            return f"{BASE_URL}/v1/videos/{task_id}/content"
        if status in {"failed", "cancelled", "canceled", "error"}:
            raise RuntimeError(data)
        time.sleep(interval)
    raise TimeoutError(task_id)

task_id = submit_sd2("一个女孩站在海边，夕阳逆光，电影感镜头")
video_url = poll_video(task_id)
print(video_url)
```
